Для написания UI-тестов можно использовать те же возможности Python, что и для написания юнит-тестов, которые создаются разработчиками. 

Любой тест должен содержать:

Входные данные.
1. Тестовый сценарий, то есть набор шагов, которые надо выполнить для получения результата.
2. Проверка ожидаемого результата.
3. Давайте обсудим, как именно можно производить проверки.
